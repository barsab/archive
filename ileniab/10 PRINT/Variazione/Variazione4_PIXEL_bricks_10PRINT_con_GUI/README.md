# Variazione con GUI

Ho inserito alcuni parametri modificabili tramite dei controlli inseriti a lato, dentro la cartella "Elemento".

Si possono modificare:
- il colore di sfondo
- la dimensione dell'elemento
- le variabile R, G, B
- la forma dell'elemento
